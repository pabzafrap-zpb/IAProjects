# Proyectos en GoogleColab de IA GEN
Proyectos varios desarrollados en Google Colab

Proyecto I: [Analisis de Oferta & Propuesta de Proyectos](https://colab.research.google.com/drive/1SW0zSUi1ZnqGebeicSdHONIfNfsMQ2Qc#scrollTo=xxx90oChhBY5)

El objetivo de este proyecto es compartirle a OpenAI un enlace a una oferta de empleo y pedirle que la analice en primer lugar para obtener los requisitos y después sugiera proyectos de análisis de datos relacionados con la oferta para aportar experiencia en la solicitud.
